simpeg.meta.MultiprocessingRepeatedSimulation.solver#

property MultiprocessingRepeatedSimulation.solver#

Numerical solver used in the forward simulation.

Many forward simulations in SimPEG require solutions to discrete linear systems of the form:

\[\mathbf{A}(\mathbf{m}) \, \mathbf{u} = \mathbf{q}\]

where \(\mathbf{A}\) is an invertible matrix that depends on the model \(\mathbf{m}\). The numerical solver can be set using the solver property. In SimPEG, the pymatsolver package is used to create solver objects. Parameters specific to each solver can be set manually using the solver_opts property.

Returns:
pymatsolver.base.Base

Numerical solver used to solve the forward problem.
