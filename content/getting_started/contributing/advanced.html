Advanced: Installing Solvers#

Pardiso is a direct solver that can be used for solving large(ish) linear systems of equations. The provided testing environment should install the necessary solvers for you. If you wish to modify pymatsolver as well follow the instructions to download and install pymatsolver.

If you open a Jupyter notebook and are able to run:

from pymatsolver import Pardiso

then you have succeeded! Otherwise, make an issue in pymatsolver.
